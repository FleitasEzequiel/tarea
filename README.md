# Instrucciones de Ejecución
## Prender servidor
Navega a la carpeta del servidor:

```bash
cd server
```
Instalar las dependencias:

```bash
npm install
```

Encender el servidor :
```bash
node server.js
```

## Cliente

Navega a la carpeta del cliente:

```bash
cd client
```
Instalar las dependencias:

```bash
npm install
```

Encender el cliente :
```bash
npm run dev
```

Abrir la página 
>https://localhost:5173
